# Campus Boilerplate Hub (scaffold)

A small collection of starter templates for campus/student projects. This scaffold demonstrates a minimal static site to browse and download simple boilerplates.

## What’s included
- index.html — landing page
- templates/basic.html — a minimal template users can copy
- styles/style.css — shared styles
- scripts/main.js — tiny JS helper
- LICENSE — MIT license
- .gitignore

## Usage
1. Copy the `campus-boilerplate-hub/` folder into your project, or serve it as static site.
2. Open `index.html` in a browser to view the hub and available templates.
3. Customize templates and contribute back via pull requests.

## Contributing
PRs welcome — add templates under `templates/` with a short README describing usage.
